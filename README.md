# feature-signal-discovery
**Goal**: Train a machine learning model to predict energy trading prices, and to find out which features have the highest impact on the model. Feed the model with additional random data from Kaggle (and possibly others). Visualize feature importance and weights with plots.

For this project we have implemented 3 different parts (all working together):
- **Prediction model**:
  * XGBoost
  * LightGBM
  * blablabla
- **Dataset Scraper**:
  * Pandas and Kaggle API
